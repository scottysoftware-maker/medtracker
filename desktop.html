What is this tool?

MedTracker Desktop Tool is a browser-based data entry and management application that allows you to enter medical information on your computer using a full keyboard and large screen, then export it to import into the MedTracker Android app.

Features

  • ✅ Add, edit, and delete people, doctors, medications, conditions, allergies, and appointments
  • ✅ Full keyboard data entry on desktop/laptop computers
  • ✅ Support for condition status (Active, Chronic, Resolved)
  • ✅ Import existing JSON backups from MedTracker app
  • ✅ Export JSON files compatible with MedTracker app
  • ✅ Works offline - no internet required
  • ✅ 100% local - your data never leaves your device
  • ✅ Compatible with Windows, Mac, and Linux

How to Use

  1. Add People First: Start by adding family members in the People tab
  2. Add Doctors: Add doctors you visit
  3. Add Medical Data: Add medications, conditions, allergies, and appointments
  4. Export JSON: Click "Export JSON" to download the data file
  5. Transfer to Phone: Email, upload to cloud storage, or copy the JSON file to your phone
  6. Import in App: Open MedTracker app → Menu → Import Database → Select the JSON file

Privacy & Security

This tool runs entirely in your browser. No data is sent to any server. All information is stored locally in your browser until you export it. When you close the browser, you can export your data or clear it using the "Clear All Data" button.
